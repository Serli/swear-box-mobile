Connexion
Bienvenue sur ISwearBox, l'application qui numérise la tirelire à gros mots !
Pour utiliser l'application, vous aurez besoin de vous connecter à un compte google.
Liste des membres
Sur la page Membres, la liste des membres enregistrés sont affichés.
Pour incrémenter la dette d'un des membres, vous devrez simplement appuyer dessus.
Pour ajouter un nouveau membre ou modifier les membres existants, deplacez vous à la page Configuration.
Configuration de l'application
Sur la page Configuration, les options générales sont configurables et les membres enregistrés sont listés.
Pour ajouter un nouveau membre ou modifier le montant de la pénalité, vous devrez simplement appuyer sur le bouton correspondant.
Pour modifier un membre existant, deplacez vous à la page configuration d'un membre en appuyant sur le membre.
Configuration d'un membre
Sur la page Configuration du membre, les options liées à la modification d'un membre y sont présentées.
Pour modifier le nom ou le prénom d'un membre ou choisir une photo de profil enregistrée sur votre smartphone, vous devrez simplement appuyer sur le champ correspondant.
Pour acquitter la dette du membre et la réinitialiser à 0, pour modifier sa photo de profil en utilisant l'appareil photo de votre smartphone ou pour supprimer ce membre, vous devrez appuyer sur le bouton correspondant.
